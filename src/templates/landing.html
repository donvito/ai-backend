Quick Start

terminal
$
git clone https://github.com/donvito/ai-backend.git
$
cd ai-backend
$
bun install
$
bun run dev
Server running at: http://localhost:3000

ðŸ”§ Environment Setup

Create a .env file in the root directory with your API configuration:

# CORS Configuration
# Supports commas and wildcards like https://*.example.com. Defaults to localhost if unset.
CORS_ORIGINS=https://*.example.com,https://app.example.com,http://localhost:3000
# OpenAI Configuration
OPENAI_API_KEY=your-openai-api-key
OPENAI_MODEL=gpt-4
# Ollama Configuration (optional)
OLLAMA_ENABLED=true
OLLAMA_BASE_URL=http://localhost:11434
OLLAMA_MODEL=llama3.2

Available Endpoints

Core AI Services

  • /api/summarize - Text summarization
  • /api/translate - Language translation
  • /api/sentiment - Sentiment analysis
  • /api/keywords - Keyword extraction

Management & Tools

  • /api/services - Service management
  • /api/ui - Swagger documentation
  • /api/jsoneditor - JSON payload editor

Supported AI Services

OpenAI

Official OpenAI SDK with structured output support. Requires API key configuration.

Ollama

Local or remote Ollama instances with JSON mode support and health checking.

The API automatically selects the best available service or you can specify service parameter in your requests.
